About song name haiku By John Wiseman

This app

  • is written in Clojure
  • runs on Heroku
  • uses the CMU pronouncing dictionary ("cmudict") for syllable information.
  • uses the Freebase API to suggest artist names and find all tracks by an artist.

The code is open source and is available on Github.
