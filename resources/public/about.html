About song name haiku By John Wiseman

This app

  • is written in Clojure
  • runs on Heroku
  • uses the CMU pronouncing dictionary AKA "cmudict" (via com.lemonodor/syllables) for syllable information.
  • uses the Freebase API (via freebase-clj to suggest artist names and to find all tracks by an artist.
  • uses Lucene for some text processing.

The code is open source and is available on Github.

If you want to see every haiku that can be made for an artist, add &all=1 to the URL, like this. If you want to see the list of known tracks and their syllable counts, add &debug=1 to the URL, like this.
